WELCOME TO COVID 19 KNOWLEDGE MANAGEMENT APPLICATION

DESIGNED BY ABHISHEK JOSHI


The steps you need to follow are:

  1. Upload the excel file with the input (in xlsx format). Click NEXT.
  2. As soon as you hit NEXT, the python based backend deployed in Heroku remote servers will run an NLP algorithm on the data to tokenise it, and will then push it to the MongoDB.
To start the process click on get started.
